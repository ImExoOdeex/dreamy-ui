## Import

- `Alert`: The main container for the alert.
- `AlertIcon`: The icon for the alert that changes based on the `status` prop.
- `AlertTitle`: The title of the alert.
- `AlertDescription`: The description of the alert.

```tsx
import { Alert, AlertTitle, AlertDescription, AlertIcon } from "@dreamy-ui/react";
```

## Usage

<Wrapper>
	<Alert status="warning">
		<AlertIcon />
		<AlertTitle>Warning</AlertTitle>
		<AlertDescription>Dream may boost Developer efficiency.</AlertDescription>
	</Alert>
</Wrapper>

```tsx
<Alert status="warning">
    <AlertIcon />
    <AlertTitle>Warning</AlertTitle>
    <AlertDescription>Dream may boost your Developer efficiency.</AlertDescription>
</Alert>
```

### Status

The `status` prop changes the color of the alert and the icon.
