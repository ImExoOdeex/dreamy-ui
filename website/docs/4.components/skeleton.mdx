## Import

- `Skeleton` - The Skeleton component.
- `SkeletonText` - The Skeleton for text content.

```tsx
import { Skeleton, SkeletonText } from "@dreamy-ui/react/rsc";
```

## Usage

Base usage of a Skeleton.

<Wrapper>
    <VStack maxW='300px' gap={4}>
        <HStack>
            <Skeleton
                boxSize="10"
                rounded={"full"}
            />
            <SkeletonText
                lines={2}
            />
        </HStack>
        <Skeleton h='150px' />
    </VStack>
</Wrapper>
